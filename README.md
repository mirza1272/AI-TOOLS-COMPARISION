## Paper Used for Comparison

**Title:** Attention Is All You Need
**Authors:** Vaswani et al.
**Year:** 2017
**Source:** arXiv:1706.03762
**Link:** https://arxiv.org/abs/1706.03762

This paper was chosen because it is publicly available, foundational to the field, and a good test for how well each model can summarize dense technical content for a general audience.
